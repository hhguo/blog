Conversational TTS Samples

  • M1: baseline model.
  • M2: M1 plus auxiliary encoder.
  • M3: M2 plus conversational context encoder.
  • The red text parts are agent (female voice), the others are user (male voice).

Dialog 3

TranscriptAudio
嗨，你们这可以点酒水吗？
当然。本次航班，您可以选择酒吧饮品，咖啡，茶或软饮料。
好。嗯，饮料的价格是多少？
我们的酒精饮料，每人五美元。其他的均为两美元，可免费续杯。
那可以点餐吗？
可以的。我们会先给您赠送一些小点心，然后您将在几个小时内享用晚餐。
今天晚餐吃什么？
我们供应鸡肉，鱼肉或牛肉盘。您可以随便选。
我可以点素食盘吗？
不好意思，先生，特殊饮食需求需要在起飞前提前预定的。

Dialog 4

TranscriptAudio
早上好。我想找一位房地产经纪人来帮我卖房子。
早上好。我很乐意为您提供帮助。那，您什么时候想卖掉您的房子？
嗯，我准备搬到我的新家，但首先得卖掉我现在的老房子。
好啊，您买房多久了呢？
嗯，两年前买的。
我想您可能知道现在市场并不那么好。如果您的房子有8年或以上，可能能卖个更好的价钱。
嗯，那如果我现在卖掉，会亏钱吗？
嗯，也不好说行情怎么会怎么变，但是您必须考虑以一个较低价格购买一个替代住房。
好的。接下来，我该怎么做呢？
好的，我想过来先看看您的房子，然后我们先起草一份合同。
